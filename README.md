# 기본

```java
import java.util.Scanner;

Scanner 변수명 = new Scanner(System.in);
int num = 변수명.nextInt();
```

- 사용자에게 입력받기 위한 라이브러리와 구조

- package = 소문자 시작

- class = 대문자 시작
- 처음에 자바프로젝트를 생성하고 Ctrl+N 눌러서 Package 만들고 그 밑에 Class 생성

# 진수

- 0B숫자 -> 2진수
  - 2진수의 1의 보수값을 구한뒤에 1을 더해주면 음수값 (1의 보수 -> 0은 1로, 1은 0으로 )
- 0숫자 -> 8진수
- 0X숫자 -> 16진수
